Description of the project: the program generates a given number of passwords and includes a smart setting for the length of the password, as well as which characters you want to include in it and which ones to exclude.

Project components:

    Integers (type int);
    variables;
    Data input / output (input() and print() functions);
    Conditional statement (if/elif/else);
    for loop;
    Writing custom functions;
    Working with the random module to generate random numbers.
